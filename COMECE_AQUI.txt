    1️⃣  Instalar Git
         → Consulte: INSTALAR_GIT.txt

    2️⃣  Fazer primeiro commit
         → Consulte: GITHUB_PUSH.txt
         → Comandos:
           git config --global user.name "Seu Nome"
           git config --global user.email "seu@email.com"
           git init
           git add .
           git commit -m "Initial commit: Task Manager"

    3️⃣  Fazer push para GitHub (opcional)
         → Criar repositório em: https://github.com/new
         → Seguir GITHUB_PUSH.txt

    4️⃣  Instalar dependências (SETUP.txt - Parte 3 e 4)
         Backend:
           cd backend
           python -m venv venv
           venv\Scripts\activate
           pip install -r requirements.txt

         Frontend:
           npm install

    5️⃣  Rodar PostgreSQL com Docker (SETUP.txt - Parte 1)
         docker-compose up -d postgres
         Verificar: docker ps

    6️⃣  Rodar Backend (SETUP.txt - Parte 6)
         cd backend
         python run.py
         Acessar: http://localhost:8000

    7️⃣  Rodar Frontend (SETUP.txt - Parte 7)
         npm run dev
         Acessar: http://localhost:5173

    8️⃣  Testar tudo
         [ ] Frontend funciona?
         [ ] Backend funciona?
         [ ] Banco dados funciona?
         [ ] Swagger funciona?
