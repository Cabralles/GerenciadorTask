  🚀 PRÓXIMOS PASSOS (ORDEM IMPORTANTE!)
================================================================================

PASSO 1: INSTALAR GIT (se ainda não tem)
  Consulte: INSTALAR_GIT.txt
  Verifique: git --version

PASSO 2: FAZER PRIMEIRO COMMIT
  Consulte: GITHUB_PUSH.txt
  Comandos:
    git config --global user.name "Seu Nome"
    git config --global user.email "seu@email.com"
    cd c:\Users\User\Downloads\Moderntaskmanagerinterface-main\Moderntaskmanagerinterface-main
    git init
    git add .
    git commit -m "Initial commit: Task Manager - React 19 + FastAPI + PostgreSQL"

PASSO 3: FAZER PUSH PARA GITHUB (opcional mas recomendado)
  1. Crie repositório em: https://github.com/new
  2. Copie a URL do repositório
  3. Execute no terminal:
    git remote add origin https://github.com/seu-usuario/task-manager.git
    git push -u origin main

PASSO 4: INSTALAR DEPENDÊNCIAS
  Consulte: SETUP.txt (Parte 3 e 4)
  
  Backend:
    cd backend
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt
  
  Frontend:
    npm install

PASSO 5: RODAR DOCKER + POSTGRESQL
  Consulte: SETUP.txt (Parte 1)
  
  docker-compose up -d postgres
  
  Verificar: docker ps (deve mostrar task_manager_db)

PASSO 6: RODAR BACKEND
  Consulte: SETUP.txt (Parte 6)
  
  cd backend
  python run.py
  
  Acessar:
    - API: http://localhost:8000
    - Swagger: http://localhost:8000/docs

PASSO 7: RODAR FRONTEND
  Consulte: SETUP.txt (Parte 7)
  
  npm run dev
  
  Acessar: http://localhost:5173

PASSO 8: TESTAR TUDO
  [ ] Frontend carrega em http://localhost:5173
  [ ] Backend responde em http://localhost:8000
  [ ] Swagger funciona em http://localhost:8000/docs
  [ ] PostgreSQL está rodando (docker ps)
